# Data

Add your data sources in a directory.

## Example

```text
data\{your-gtfs-provider}\agency.txt
data\{your-gtfs-provider}\calendar_dates.txt
data\{your-gtfs-provider}\feed_info.txt
data\{your-gtfs-provider}\pathways.txt
data\{your-gtfs-provider}\routes.txt
data\{your-gtfs-provider}\shapes.txt
data\{your-gtfs-provider}\stop_times.txt
data\{your-gtfs-provider}\stops.txt
data\{your-gtfs-provider}\trips.txt

data\{your-other-gtfs-provider}\agency.txt
data\{your-other-gtfs-provider}\calendar_dates.txt
data\{your-other-gtfs-provider}\feed_info.txt
data\{your-other-gtfs-provider}\pathways.txt
data\{your-other-gtfs-provider}\routes.txt
data\{your-other-gtfs-provider}\shapes.txt
data\{your-other-gtfs-provider}\stop_times.txt
data\{your-other-gtfs-provider}\stops.txt
data\{your-other-gtfs-provider}\trips.txt
```
